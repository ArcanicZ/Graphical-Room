To run the server:

1. Download and Install NodeJS.
2. Install the http-server package from npm by going in command prompt and entering 'npm install -g http-server'.
3. Change to the directory that the code is in so in my case it would be within the dpnx39_CG folder.
4. type 'http-server' into the command prompt.
5. Now on a browser enter the url that is provided within the command prompt. It will usually be http://127.0.1:8080.
6. Click on viewCanvas.html and it should now open up the program.
